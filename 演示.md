# BBS_Talk 匿名论坛 — 验收演示

> 项目：轻量级多分区匿名论坛  
> 技术栈：Spring Boot 2.7 + MyBatis + Thymeleaf + MySQL + Bootstrap 3 + 阿里云 OSS  
> 启动：运行 `run.bat` / `run.sh` / `run.py`，访问 `http://localhost:8080`

---

## 演示流程（按顺序展示）

### 1. 首页浏览
- 打开 `http://localhost:8080`，看到**动态三角形背景** Canvas 动画
- 帖子列表：每帖显示标题、分类徽标、内容预览（4 行截断）、封面缩略图、作者头像+昵称、时间、回复数、收藏数
- **分类筛选**下拉框：全部 / 娱乐 / 技术 / 美食 / 旅游 / 问题
- **分页**：每页 4 帖，可翻页
- **置顶帖子**（`is_sticky=1`）置顶显示，带"置顶"标记
- 导航栏右侧铃铛图标显示未读通知数（红色角标）

### 2. 用户注册
- 点击导航栏"注册"，填写用户名、密码、确认密码
- 注册成功后自动登录，右上角显示用户头像和菜单

### 3. 用户登录 / 登出
- 点击"登录"弹出模态框，输入用户名 + 密码
- 登录后导航栏切换为**用户菜单**：发帖、我的帖子、我的收藏、个人管理、退出
- 点击退出，回到未登录状态

### 4. 发布帖子
- 登录后点击"发帖"按钮
- 填写：标题、选择分类（5 类之一）、正文（支持 HTML）
- 可选上传**封面图片**（阿里云 OSS 存储）
- 选择"问题"分类时，额外出现**悬赏积分**输入框（1–10 分），从自己账户扣除
- 发布成功，首页刷新可见新帖

### 5. 帖子详情 & 回复
- 点击任意帖子进入 `/post/{pid}.html`
- 展示：封面图、标题、正文、作者信息、时间、回复数、收藏数
- **回复帖子**：底部回复按钮 → 弹出模态框输入内容 → 提交
- 回复列表分页显示（每页 10 条），带楼层号
- **回复删除**：楼主、回复作者、管理员可删
- **回复置顶**（仅楼主操作）：点击"置顶回复"，该回复升至最前；再次点击取消置顶
  - 若为"问题"分类帖子，置顶回复时将**悬赏积分**转给回复作者；取消置顶则扣回
- 点击作者头像/昵称 → 弹出**用户信息浮窗**（AJAX 获取 `/user/profile/{uid}`）

### 6. 实时通知 — 回复通知
- 用账号 A（如 u1）登录，用账号 B（如 u2）在 A 的帖子下回复
- 账号 A 导航栏铃铛出现红色角标（**WebSocket 实时推送**，无需刷新页面）
- 点击铃铛 → 弹出下拉面板显示未读通知，点击通知跳转到帖子详情

### 7. 帖子管理
- **我的帖子** `/myPosts.do`：表格展示自己所有帖子，可**查看**、**编辑正文+封面**、**删除**
- **删除帖子**：级联删除该帖下所有回复和收藏记录

### 8. 收藏功能 & 收藏通知
- 首页/帖子页点击 ♡ 按钮 → **收藏帖子**（按钮变红实心）
- 再次点击 → **取消收藏**
- **我的收藏** `/favorite/list`：分页展示收藏的帖子，同样支持分类筛选
- 每帖可"取消收藏"或"删除"（楼主/管理员）
- 收藏他人帖子时，帖子作者收到**收藏通知**（WebSocket 实时推送）

### 9. 关注/粉丝 & 关注通知
- 点击任意用户头像/昵称 → 弹出**用户信息浮窗**，底部出现「关注」按钮
- 关注后按钮变为「取消关注」
- **我的关注** `/follow/list?type=following`：分页展示已关注用户
- **我的粉丝** `/follow/list?type=fans`：分页展示关注我的用户
- 关注他人时，被关注者收到**关注通知**（WebSocket 实时推送）

### 10. 通知中心
- 点击导航栏铃铛 → 进入 `/notification/page` 通知中心
- **分类标签**：全部 / 回复 / 收藏 / 关注 / 系统
- 通知卡片显示：类型徽标（回复=蓝、收藏=红、关注=绿、系统=灰）、时间、内容
- **未读通知**高亮黄色边框，内容加粗
- 点击通知：
  - 回复/收藏类型 → 跳转到对应帖子
  - 关注/系统类型 → 弹出用户信息浮窗
- 操作：**标为已读**（单个）、**全部已读**、**删除**（系统通知不可删）

### 11. 个人管理
- **个人中心** `/person.do`：三栏折叠面板
  - **修改密码**：旧密码 → 新密码 → 确认新密码（前端 JS 校验一致性）
  - **个人资料**：头像预览 → 点击上传 → Cropper.js 裁剪 → 上传至 OSS；编辑电话、职业、地址
  - **注销账号**（普通用户）：红色警告提示 → 确认后级联删除该用户所有回复/帖子/收藏

### 12. 管理员后台
- 管理员账号：`admin / 111222333`
- 登录后导航栏出现"后台管理"入口 → `/host.do`
- **用户管理**表格：UID、用户名、注册时间、状态（正常/禁言）
- 操作：**禁言/解禁**（被禁言用户无法发帖回复）、**删除用户**（级联删除所有数据）

### 13. 管理员 — 全局通知
- 后台页面底部"全局通知"面板：输入通知内容 → 点击"发送全局通知"
- 所有在线用户立即收到 WebSocket 推送，离线用户登录后可在通知中心查看
- 系统通知显示灰色"系统"徽标，不可删除，点击弹出管理员信息浮窗

### 14. 帖子置顶管理（管理员）
- 首页/帖子页管理员可见"置顶"/"取消置顶"按钮
- 置顶帖子在首页始终排在最前

### 15. 彩蛋功能（pid=12 "能玩一辈子音游吗"）
- 访问 `/post/12.html`：
  - **音乐播放器**：两首曲目（CHAIN2NITE、Cosmogyral），带封面、进度条
  - **"狠狠批评"按钮**：点击进入 10 步多选吐槽小游戏
    - 每步一个搞笑问题 + 趣味选项
    - 全部答完 → 全屏祝贺 → 跳转视频页 `/video/12`
    - 答错 → 弹窗提示后关闭
  - **视频页** `/video/12`：自动播放 OSS 视频，自定义播放/暂停/进度条 → 播放结束跳回首页

### 16. 异常处理
- 访问不存在的页面 → 错误页，显示错误信息，5 秒倒计时返回首页
